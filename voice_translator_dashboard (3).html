Voice Translation Suite 사용 방법
📋 사용 방법
01
왼쪽 사이드바에서 번역 언어를 선택합니다. 🇯🇵 일본어 / 🇺🇸 영어 / 🇫🇷 불어 / 🇨🇳 중국어
02
녹음 시작 버튼 클릭 → Silero VAD가 실시간 음성 구간을 감지합니다.
03
한국어로 말하면 VAD가 speech_start를 감지하고 자동 버퍼링이 시작됩니다.
04
말을 멈추면 speech_end를 감지하고 Google STT로 한국어 텍스트를 추출합니다.
05
deep_translator가 선택 언어로 번역하고, ▶ 듣기 버튼으로 TTS 음성을 재생합니다.
06
결과는 🗂 히스토리에 자동 저장됩니다.

🔧 오류 해결
Q
음성이 인식 안 됨 → energy_threshold를 낮추거나 마이크 볼륨을 높이세요.
Q
다른 말이 붙어 인식됨 → threshold=0.7로 VAD 감도를 높이세요.
Q
TTS 재생 안 됨 → 브라우저 오디오 권한을 허용했는지 확인하세요.
